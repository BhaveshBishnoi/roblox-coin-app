# Roblox Coin Giver App

This is a React Native / Expo app designed with a dark, premium aesthetic.

## Features
- **Daily Coins**: Claim rewards every 24h.
- **Lucky Wheel**: Spin to win coins.
- **Scratch Cards**: Scratch to reveal prizes.
- **Quiz**: Answer Roblox trivia.
- **Flip Cards**: Test your luck.
- **Tips & Tricks**: Learn how to succeed in Roblox.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the app:
   ```bash
   npx expo start
   ```

## Note
This app includes a simulated "Ad Wall" feature where actions trigger an in-app browser redirect to `https://games.biographydata.org` as per requirements.
